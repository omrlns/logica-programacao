# 📘 Lógica de Programação e Algoritmos com JavaScript

Este repositório contém uma série de exercícios e exemplos práticos voltados para o aprendizado da lógica de programação utilizando a linguagem JavaScript. 
Ideal para iniciantes que desejam desenvolver raciocínio lógico e estruturar algoritmos de forma clara e eficiente.

---

## 🚀 Tecnologias Utilizadas

- **JavaScript**
- **HTML5**
- **CSS3**

---

🎯 Objetivos

- Aprimorar a lógica de programação.
- Resolver problemas com algoritmos simples.
- Ganhar fluência com estruturas fundamentais do JavaScript, HTML e CSS.
- Preparar a base para estudar estruturas de dados e algoritmos avançados.

---

🧠 Para quem é este repositório?

- Iniciantes em programação.
- Estudantes que estão aprendendo JavaScript, HTML e CSS.
- Pessoas que querem praticar resolução de problemas com algoritmos básicos.
